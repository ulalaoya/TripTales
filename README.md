# TripTales — יומן המסע המשפחתי

אפליקציית ווב משפחתית לתיעוד ותכנון טיולים. עברית-first (RTL), מובייל-first, עם החלפת שפה חיה he↔en.

## הרצה מקומית

```bash
npm install
npm run dev      # http://localhost:5173
```

פקודות נוספות:

```bash
npm run test     # Vitest — 91 בדיקות יחידה
npm run build    # בנייה לפרודקשן (tsc --noEmit && vite build)
npm run preview  # תצוגה מקדימה של הבנייה
```

### כניסות דמו

הנתונים נזרעים אוטומטית בהרצה ראשונה (נשמרים ב-localStorage):

| תפקיד | טלפון |
|---|---|
| מבוגר (אבא) | `0501234567` |
| ילד (בן) | `0527654321` |

## תכולה

**מסכים:** בית (טיול קרוב, גישה מהירה, הבא בתור, הדופק המשפחתי) · טיולים (טאבי סטטוס) · תכנון טיול · יום · אלבום · רגע חדש · רשימת ציוד · משפחה.

**תפקידים:** מבוגר יוצר ועורך טיולים וכותב יומן; ילד מעלה תמונות שממתינות לאישור הורה, ומגיב באימוג'ים. ההרשאות נאכפות גם ב-UI וגם בראוטים ובחנות הנתונים.

**רשימת ציוד:** קבוצות עם אימוג'י, אחראי לכל פריט, וטבעת התקדמות שמזינה את אחוז "התכנון הושלם" במסך הבית. כולם יכולים לסמן פריטים; רק מבוגרים מוסיפים ומוחקים.

## סטאק

React 18 · Vite · TypeScript · Tailwind · Zustand (persist → localStorage) · React Router · Vitest.

הלוגיקה הטהורה יושבת ב-`src/lib/` (טלפון, הרשאות, ריאקציות, הזמנות, מפות, צ'קליסט, סטטוס טיול) ומכוסה בבדיקות יחידה.

## עיצוב

שפת "Coral Journey": כרטיסים לבנים על קנבס קריר, אקסנט קורל `#ff6b66`, פונט Assistant, ניווט תחתון צף. כל הטוקנים ב-`:root` בקובץ `src/index.css`.

## פריסה
